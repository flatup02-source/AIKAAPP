# 使用量監視と自動停止機能の実装

## 概要

各サービスの無料プランの上限に近づいたら自動的にストップする機能を実装します。

## 監視対象サービスと上限

### 1. LINE Messaging API
- **上限**: 500通/月
- **警告閾値**: 400通（80%）
- **停止閾値**: 480通（96%）

### 2. Dify API（クラウド版）
- **上限**: 200回/月
- **警告閾値**: 160回（80%）
- **停止閾値**: 190回（95%）

### 3. Google Cloud Video Intelligence API
- **上限**: 1,000分/月
- **警告閾値**: 800分（80%）
- **停止閾値**: 950分（95%）

### 4. Google Cloud Storage
- **上限**: 5GB
- **警告閾値**: 4GB（80%）
- **停止閾値**: 4.8GB（96%）

### 5. Firebase Authentication
- **上限**: 50,000認証/月
- **警告閾値**: 40,000認証（80%）
- **停止閾値**: 48,000認証（96%）

## 実装内容

1. 使用量監視API
2. 使用量記録機能（Firestore）
3. アラート機能（メール/LINE通知）
4. 自動停止機能
5. n8n/Make.comワークフローでの使用量チェック
